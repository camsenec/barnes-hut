# A Barnes-Hut Simulation
divides a 2D field into the minimum number of subfields so that exactly one point is in a subfield

## Environment
MacOS 10.13.* >=

## Requirements
Python 3.8.* 

## Usage
### MacOS
```
$ make
$ make burnes_hut
```

### Linux
```
$ make
$ python burnes_hut.py
open the output.png with your favorite application
```
